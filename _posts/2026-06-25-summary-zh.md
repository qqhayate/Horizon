## [Nub: 通过预加载钩子为 Node.js 提供类似 Bun 的一体化工具包](https://github.com/nubjs/nub) ⭐️ 8.0/10

Colin McDonnell 发布了 Nub，这是一个为 Node.js 提供的一体化工具包，通过预加载钩子添加类似 Bun 的开发体验（转译器、模块解析、polyfills），无需替换 Node 运行时。Nub 使用打包为 Node-API 插件的 oxc 转译器，并注入 Worker 和 Temporal 等 API 的 polyfills。 Nub 提供了一种轻量级的方式，在 Node.js 生态系统中享受 Bun 的开发体验优势，可能吸引不愿切换运行时的开发者。它弥合了 Node.js 与新型一体化工具之间的差距，在不放弃现有基础设施的情况下提高生产力。 Nub 使用原版 Node 运行，通过--require 预加载钩子注册 oxc 驱动的转译器和模块解析钩子，并注入 polyfills。它完全是附加性的，意味着你的代码最终在 Node 的引擎和标准库实现上运行。

hackernews · colinmcd · 6月24日 14:14 · [社区讨论](https://news.ycombinator.com/item?id=48660267)

**背景**: Bun 是一个一体化 JavaScript 运行时，集成了转译器、包管理器和测试运行器，提供快速性能和原生 TypeScript 支持。Node.js 传统上需要单独的工具进行 TypeScript 编译和模块打包，尽管最近的版本添加了实验性的 TypeScript 支持。Oxc 是一个用 Rust 编写的高性能 JavaScript 工具链，能够解析、转换和压缩代码。Temporal API 是一个现代的 JavaScript 日期/时间 API，目前处于 stage 3 阶段，尚未在 Node 中原生支持。
